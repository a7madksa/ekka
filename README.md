# إكة — Ekka

**فن البلوت السعودي** | The Art of Saudi Baloot

---

## عن التطبيق

إكة هي لعبة بلوت سعودية أصيلة، مصممة للعب المحلي بدون اتصال بالإنترنت. العب ضد ذكاء اصطناعي متقدم يحاكي أسلوب اللعب الحقيقي.

## About

Ekka is an authentic Saudi Baloot card game built for offline play. Challenge an advanced AI that mirrors real gameplay strategy — bidding, doubling, declarations, and all.

### Features
- Hukum Awal, Hukum Thani, Sun, and Ashkal contracts
- Full doubling chain: Double → Triple → Quadruple → Qahwa
- Declarations: Tierce, Quarte, Quint, Eka, Balot
- Detailed round-by-round statistics
- Clean Arabic-first UI
